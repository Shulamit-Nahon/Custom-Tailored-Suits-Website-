# Custom Tailored Suits Website 

This is a simple static website created for showcasing custom-tailored suits. The website is built using only HTML and CSS, without any JavaScript for responsiveness.

## Table of Contents

- [Overview](#overview)
- [Features](#features)
- [Pages](#pages)
- [Installation](#installation)
- [Screenshots](#screenshots)

## Overview

The Custom Tailored Suits website is designed to provide users with information about a fictional custom suit business. The website features a clean and elegant design, emphasizing simplicity and ease of navigation.

## Features

- **Static Website**: Built with pure HTML and CSS.
- **Navigation Bar**: Includes links to all the main sections of the site.
- **Responsive Design**: The site layout is designed to work on various screen sizes, though no JavaScript is used for additional responsiveness.
- **Informational Pages**: Each page provides relevant information about the business.

## Pages

- **Home**: The landing page with an introduction to the business.
- **About Us**: A page that provides details about the company, its history, and its mission.
- **Types**: An overview of the different types of suits offered.
- **FAQ**: Frequently Asked Questions to help users find answers to common inquiries.
- **Contact**: A page with contact information and a form for users to get in touch.
